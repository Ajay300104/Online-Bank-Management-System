                                     Online Bank Management System

This is a basic console-based online bank management system implemented in Java. The system allows users to create accounts, log in, perform transactions such as deposits, withdrawals, and transfers, and view transaction history.
Features
•	Account Management:
o	Create Account
o	View Account Details
o	Delete Account
•	Transactions:
o	Deposit
o	Withdraw
o	Transfer Funds
o	View Transaction History
•	User Authentication:
o	Login
o	Logout
